# Shape Puzzle Playable Ad

### Initial Setup

**Clone Repository**

    git clone git@github.com:GroveA/burny-test-task.git
    cd burny-test-task

**Install Dependencies**
npm install

### Development

    npm start

    - Opens development server at http://localhost:8080

### Build html file

    npm run build

    - Creates output html file in `dist` folder
